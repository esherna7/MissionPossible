# MissionPossible Backend

This is the Java Spring Boot backend for the MissionPossible application.  
It provides REST APIs that the React frontend consumes.

---

## Prerequisites

Make sure you have installed:

- **Node.js** (v18+)
- **npm** (comes with Node.js)
- **Java JDK** (v17+ recommended, Java 25 supported)
- **Maven** (included with Spring Boot wrapper `mvnw`)
- **PostgreSQL**
