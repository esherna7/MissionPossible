# MissionPossible Frontend

This is the React + Vite frontend for the MissionPossible application.  
It communicates with the Java Spring Boot backend running at `http://localhost:8080`.

---

## Prerequisites

Make sure you have installed:

- **Node.js** (v18+)
- **npm** (comes with Node.js)
- **Java JDK** (v17+ recommended, Java 25 supported)
- **Maven** (included with Spring Boot wrapper `mvnw`)
- **PostgreSQL**
